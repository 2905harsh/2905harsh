## Featured Projects

### 🍔 [Bites](https://github.com/2905harsh/bites)
High-performance REST API with a **cache-first architecture** using Redis advanced features — Sorted Sets, RediSearch, Bloom Filters, and RedisJSON — for pagination, full-text search, and cache-penetration protection.

`Node.js` `Express.js` `TypeScript` `Redis`

---

### 🤖 [Project Planner MCP](https://github.com/2905harsh/project-planner-mcp)
A remote **MCP server** deployed on Cloudflare Workers that lets AI assistants manage project data through natural conversation. Uses Cloudflare Durable Objects for stateful session handling, Zod for schema validation, and supports both SSE and Streamable HTTP.

`Cloudflare Workers` `MCP` `Durable Objects` `Zod` `TypeScript`

---

### 🔔 [Real-Time Notifications](https://github.com/2905harsh/real-time-notifications)
Real-time notification system using **Socket.IO** with username-to-socketId mapping for precise one-to-one targeted delivery. Handles the full WebSocket lifecycle including offline client detection.

`Node.js` `Socket.IO` `WebSockets`
